КОРТЕЖ
Кортеж неизменяемый
Кортеж может содержать в себе ссылки на объекты разного типа


T = 1, 2, 3, 4, 5
print(type(T))   # <class 'tuple'>

a, b, c, d, e = T  #   количество элементов должно совпадать с длиной кортежа

По элементам
a = T[0]
b = T[1]

a,b,*other = T   #  a и b первые два элемента остальное в other   other - ссылается на новый кортеж


ВЫВОД Кортежа

print(T)   # (1, 2, 3, 4, 5)
print(T[0], T[1], T[2], T[3], T[4])  # 1 2 3 4 5
print(*T)  # * - функция поэлементной распаковки и print выводит поэлементно
